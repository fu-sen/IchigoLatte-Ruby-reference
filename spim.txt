spim
type R 0.5.0～
-----------------------------------------------------------
[ 機能 ]
SPI のモードを設定します。

[ 書式 ]
spim(<モード>)

[ 例 ]
spim(0)

[ 解説 ]
<モード> は次のとおりです。

       -1 SPI オフ
       0  CPOL 0  CPHA 0
       1  CPOL 0  CPHA 1
       2  CPOL 1  CPHA 0
       3  CPOL 1  CPHA 1

このコマンド実行によりビデオ画面は非表示となります。
